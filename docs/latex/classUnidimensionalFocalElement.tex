\subsection{Detailed Description}
1D \hyperlink{classFocalElement}{Focal\+Element} using an integer ID as source of information. 

The ID binary representation encodes the singleton hypotesis which are included in the current set. E.\+g. ID=5 -\/$>$ 101 -\/$>$ \{h1,h3\} focal element.
